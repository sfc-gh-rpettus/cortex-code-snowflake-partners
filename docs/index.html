About This Project

Why these resources?

Cortex Code understands and works across the entire data ecosystem. These references are designed for Snowflake partners and ISVs who are building with and on the Snowflake platform and want to use Cortex Code effectively in their workflows.

What's covered:

  • Snowflake-native tools — #TABLE syntax, /sql, Cortex Analyst, Semantic Views
  • Data ecosystem workflows — native Airflow and dbt plugin integration
  • Skills system — progressive disclosure architecture for reusable instruction sets
  • MCP integration — connecting external tools via Model Context Protocol
  • Agents & subagents — general-purpose, Explore, Plan, dbt-verify, and more
  • Project setup — AGENTS.md, hooks, configuration, and safety guardrails

All content is verified against the latest Cortex Code documentation. These are living documents and will be updated as new features ship.
